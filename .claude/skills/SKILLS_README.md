# Skills Index

This directory contains Claude Code skills that provide specialized functionality for common development tasks.

## Available Skills

### Documentation

- **[docstring](./docstring)** - Add documentation to functions and classes
- **[markdown-fmt](./markdown-fmt)** - Format and clean up markdown files
- **[readme](./readme)** - Generate or update a README file

### Exploration & Research

- **[explore](./explore)** - Quickly explore and understand a codebase
- **[research](./research)** - Research external documentation, libraries, or technical topics
- **[summarize](./summarize)** - Summarize code, documentation, or findings into a concise format

### Code Generation

- **[api-spec](./api-spec)** - Generate OpenAPI specifications for REST APIs
- **[auth](./auth)** - Guide for integrating authentication providers

### Git & GitHub

- **[changelog](./changelog)** - Generate a changelog from git commits
- **[commit](./commit)** - Create well-formatted git commits following conventional commit standards
- **[pr-create](./pr-create)** - Create a pull request with a well-structured description
- **[pr-review](./pr-review)** - Review a pull request for quality, security, and correctness
- **[ticket](./ticket)** - Work on a ticket/issue from start to finish

### MCP Development

- **[mcp-auth](./mcp-auth)** - Add authentication to an MCP server
- **[mcp-init](./mcp-init)** - Scaffold a new MCP server project
- **[mcp-resource](./mcp-resource)** - Add a new resource to an MCP server
- **[mcp-test](./mcp-test)** - Test an MCP server locally
- **[mcp-tool](./mcp-tool)** - Add a new tool to an MCP server

### Workflow & Best Practices

- **[plan](./plan)** - Create a structured implementation plan before writing code
- **[refactor](./refactor)** - Safely refactor code without changing behavior
- **[tdd](./tdd)** - Follow test-driven development

## Usage

Skills are invoked using the `/skill-name` command in Claude Code. Each skill provides specific instructions and context to help with common development tasks.

For more information about skills, see the [Claude Code Skills documentation](https://code.claude.com/docs/en/skills).
